# acos
Игра: OFP v1.0
## Описание
Возвращает арккосинус числа, в градусах
## Синтаксис
```
acos <число>;
```
Аргументы:
* <число> NUMBER - значение синуса в диапазоне [-1.0, 1.0]
Результат:
NUMBER - арккосинус переданного значения, в градусах
## Пример(ы)
```
acos 0.5;   //60
```

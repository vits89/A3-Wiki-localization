# ABS
Игра: OFP v1.00  
Аргументы: локальные  
Эффект: локальный  
## Описание
Абсолютное значение числа
## Синтаксис
```
abs <число>
```
### Параметр(ы)
<число> NUMBER - любое число
### Результат
NUMBER - абсолютное значение числа, переданного в параметрах команды
## Пример(ы)
```
abs -10;  //Результат: 10
```
## Примечания
Нет

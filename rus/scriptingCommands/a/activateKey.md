# activateKey

Игра: ArmA v1.0

## Описание

Активирует ключ с данным именем для текущего профиля

## Синтаксис

```SQF
activateKey <ключ>
```

Аргументы:

* <ключ> STRING - имя ключа

Результат:
NOTHING

## Пример(ы)

```SQF
activateKey "Mission04Key";
```

## Дополнительная информация

* Ключи в основном используются для разблокировки миссий или кампаний
